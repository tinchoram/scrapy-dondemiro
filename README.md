¿Dónde Miro?
Esta es la pregunta que me hacen frecuentemente mis amig@s y conocido@s cuando no encuentran Películas o Series en las plataformas habituales. En lugar de recomendarles un sitio en particular les recomiendo varios!
“¿Dónde Miro?” es una web que recopila mediante Web Scraping en Python 🐍 el listado de películas y series  de los sitios mas conocidos, evitando que tengas que visitar cada uno para realizar una Búsqueda!
